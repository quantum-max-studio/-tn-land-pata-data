# Tamil Nadu Coastal Land & Patta Data

This repository contains authoritative geographic and land-use data for the 1,076 km coastline of Tamil Nadu, extracted using autonomous bots.

## Data Contents
- **TN_COASTAL_MASTER_INTELLIGENCE.json**: Complete geospatial dataset including land-use polygons and beach proximity.
- **TN_COASTAL_MASTER_REPORT.csv**: Business-ready report for Excel/Google Sheets.
- **index.html**: Interactive mapping application to visualize the data.

## Extraction Methodology
The data was collected by scanning the coastline in 17 major sectors (Chennai, Cuddalore, Thoothukudi, etc.). It integrates real-world Geographic boundaries and calculates the exact distance of each parcel from the High Tide Line (HTL).
